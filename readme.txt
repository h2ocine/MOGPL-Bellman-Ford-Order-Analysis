Ce projet explore l'impact de l'ordre des sommets sur l'efficacité de l'algorithme de Bellman-Ford pour résoudre le problème des plus courts chemins dans un graphe orienté pondéré. L'objectif est d'affiner l'estimation du nombre d'itérations nécessaires pour que l'algorithme converge en examinant l'influence de l'ordre des sommets sur la propagation de l'information et les chemins les plus courts. L'analyse permettra de mieux comprendre les facteurs qui affectent les performances de l'algorithme dans différents contextes.

Contenu du dossier
projetMOGPL.ipynb: Ce fichier est le fichier principal du projet. Il contient le code du programme ainsi que tous les tests réalisés.
Graph.py: Ce fichier contient la classe Graph, qui est utilisée dans le programme principal pour représenter et manipuler les graphes.
util.py: Ce fichier contient des fonctions utilitaires utilisées dans le programme principal.
rapport.pdf: Ce fichier contient le rapport du projet, décrivant en détail les objectifs, les méthodes utilisées et les résultats obtenus.

Utilisation du programme
Pour exécuter le programme, vous devez disposer d'un environnement Python avec les bibliothèques nécessaires installées. Le fichier principal du programme est projetMOGPL.ipynb, qui peut être ouvert et exécuté dans un environnement Jupyter Notebook ou dans tout autre environnement compatible avec les notebooks Jupyter.
